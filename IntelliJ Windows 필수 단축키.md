# IntelliJ Windows 필수 단축키 치트 시트
윈도우 환경에서 자바/스프링 개발할 때 손에 익혀두어야 하는 인텔리제이 필수 단축키 요약입니다.

## 코드 자동 생성
* `psvm` + `Enter` : `public static void main` 메서드 생성
* `sout` + `Enter` : `System.out.println()` 생성
* `Alt` + `Insert` : 생성자(Constructor), Getter/Setter, toString 자동 생성

## 코드 정리 및 편집
* `Ctrl` + `Alt` + `L` : 지저분한 코드 줄 바꿈 및 정렬 자동 맞춤
* `Ctrl` + `Alt` + `O` : 안 쓰는 import 문 자동으로 싹 삭제
* `Ctrl` + `D` : 현재 줄 그대로 아래에 복사
* `Ctrl` + `Y` : 현재 줄 한 줄 삭제
* `Shift` + `F6` : 변수나 메서드 이름 전체 일괄 변경

## 찾기 및 실행
* `Shift` 두 번 연타 : 프로젝트 내 전체 파일 및 클래스 검색
* `Ctrl` + `Shift` + `F` : 전체 파일에서 특정 텍스트(문자열) 검색
* `Shift` + `F10` : 이전에 실행했던 서버/메인 메서드 재실행
