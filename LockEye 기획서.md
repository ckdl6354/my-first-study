# [기획] 시선 고정형 몰입 생산성 풀스택 웹 서비스: LockEye

> "공부나 코딩 도중 발생하는 브라우저 화면 이탈을 강제로 막아, 현대인들의 온전한 몰입을 돕는 자바 풀스택 기반의 생산성 웹 서비스입니다."

## 1. 서비스 개요
- **서비스 이름:** LockEye (락아이)
- **기획 의도:** 기존의 스마트폰 타이머 앱들과 달리, PC로 웹 개발 공부를 할 때 발생하는 '브라우저 탭 이탈(유튜브, 웹툰 등)'을 직접적인 방해 요소로 정의했습니다. 개발 공부 전용 브라우저 화면을 강제로 유지하게 함으로써 몰입 환경을 극대화합니다.

## 2. 핵심 기능 (풀스택 구현 상상도)

### Frontend (사용자 화면단)
- **집중 타이머:** 뽀모도로 타이머 인터페이스 제공 (30분 집중 / 5분 휴식).
- **시선 이탈 감지:** 브라우저의 Page Visibility API를 활용하여 사용자가 다른 탭으로 넘어가거나 브라우저 창을 내리면 이를 실시간으로 감지.
- **패널티 시스템:** 화면 이탈 후 복귀 시 '자바 핵심 개념 OX 퀴즈' 강제 팝업. 정답을 맞히면 타이머가 유지되나, 오답이거나 제한 시간 초과 시 타이머를 강제 초기화하는 지능형 패널티 시스템 구현.

### Backend (서버 및 데이터베이스단)
- **집중 세션 관리:** 사용자가 화면 이탈 없이 30분 집중을 무사히 완료하면 해당 데이터(시간, 날짜)를 자바 스프링(Spring Boot) 서버로 전송하여 보관.
- **도파민 통계 데이터:** 사용자가 하루에 총 몇 분을 집중했는지, 집중 도중 딴짓을 시도해 퀴즈 패널티를 받은 횟수가 몇 번인지 데이터를 정제하여 마이페이지 등에 제공.

## 3. 예상 기술 스택
- **Frontend:** HTML5, CSS3, JavaScript (Page Visibility API 기반 시선 이탈 감지 및 자바/스프링 핵심 개념 OX 퀴즈 시스템 구현)
- **Backend:** Java 17, Spring Boot, Spring Data JPA
- **Database:** MySQL (회원 정보 및 일일 집중 로그 저장)


## 4. 이 프로젝트를 통해 성장하고 싶은 점
- 웹 브라우저가 제공하는 기본 API(Visibility API)와 프론트엔드 이벤트를 깊이 있게 다루는 방법 학습.
- 프론트엔드에서 수시로 발생하는 집중 데이터 요청을 백엔드 서버가 안정적으로 받아 처리하고 데이터베이스 구조를 효율적으로 설계하는 풀스택 데이터 흐름 마스터.
