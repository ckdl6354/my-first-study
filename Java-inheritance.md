상속 (Inheritance)
기존 클래스(부모)의 필드와 메서드를 새로운 클래스(자자식)가 물려받아 재사용하는 문법

### 핵심 개념
* **`extends` 키워드**: 자식 클래스가 부모 클래스를 상속받을 때 사용
* **단일 상속**: 자바는 다중 상속을 허용하지 않아 하나의 부모 클래스만 상속
* **`super` 키워드**: 부모 클래스의 멤버(필드, 메서드, 생성자)를 참조할 때 사용

### 오버라이딩 (Overriding)
* 부모에게 물려받은 메서드를 자식 클래스에 맞게 **재정의**
* 메서드 이름, 매개변수, 반환 타입이 완전히 일치해야함
* `@Override` 어노테이션을 붙여 컴파일러에게 체크를 요청하는 것이 안전

## 예외 처리 (Exception Handling)
프로그램 실행 중 발생할 수 있는 예기치 못한 에러(예외)를 처리하여 프로그램의 비정상 종료를 막는 문법
### `try - catch - finally` 구조
* **`try` 블록**: 예외가 발생할 가능성이 있는 코드를 작성
* **`catch` 블록**: 예외가 발생했을 때 이를 잡아 안전하게 처리하는 코드를 작성
* **`finally` 블록**: 예외 발생 여부와 상관없이 **무조건 실행**되는 코드를 작성
* (예: 리소스 반납 등)
---
### 예외 던지기 (`throws` vs `throw`)
* **`throws`**: 메서드 선언부에 사용하며, 발생한 예외를 내가 처리하지 않고 **호출한 곳으로 떠넘길 때** 사용
* **`throw`**: 개발자가 **고의로 예외를 발생**시킬 때 사용
* (예: `throw new IllegalArgumentException();`)
